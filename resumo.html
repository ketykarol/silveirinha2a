Resumo Front-End

    As linguagens de Front-End são utilizadas na criação de interfaces, ou seja, das partes de uma página com as quais o usuário pode interagir e executar ações.

  • A tag HTML é uma linguagem de marcação utilizada para contruir páginas na web.

  • O css é uma forma de adicionar estilos a uma página da web.

  • As tag HTML são usadas para informar o navegador a estrutura do site.

    • Tags do HTML

  • A tag "!DOCTYPE html" ela informa a versão do HTML que está sendo utilizada no documneto

  • A tag "BODY" ela representa o corpo do documento.

  • A tag "HEAD" ela delimita o cabeçalho do documento, ela transmiti diversas informações úteis para o documento.

  • A tag "TITLE" ela define o título da página.

  • A tag "META" ela permite inserir metados ao documento.

  • A tag "H1" ela define o titulo de maior valor, já a tag "H6" demenor valor.

  • A tag "P" ela é a principal tag dos textos, ela forma um parágrafo.

  • A tag "IMG" ela serve para colocar uma imagem no nosso documento

  • A tag "STYLE" ela é o estilo do documento.

    • Get e Post diferenças

  • O método GET cria uma sequência de consulta e a acresenta a URL do script no servidor que manipula a solicitação.

  • O método POST cria um par nome/valor que são passados no corpo da mensagem de pedido HTTP.
